## 🚀 功能概述
- 自动检查并更新Worker脚本
- 支持定时和手动触发更新
- 双渠道更新（稳定版/测试版）
- 自动代码反混淆处理
- **智能配置记忆** - 自动记住上次的参数选择

## 📖 使用说明
### 🤖 自动模式
- 每日北京时间02:00自动检查更新
- 使用上次缓存的工作参数配置

### 👆 手动模式
1. 进入GitHub Actions页面
2. 选择"Worker Auto-Updater"工作流
3. 点击"Run workflow"并选择参数：
   - 更新模式（自动/手动）
   - 发布渠道（稳定版/测试版）
   - 是否反混淆代码（true/false）
   > 💡 留空参数将使用上次配置
